# Telegram News Bot

Телеграм-бот для получения новостных заголовков из базы данных с поддержкой подписок и автоматической рассылки.

## Возможности

- получение новостей из БД
- просмотр новостей по источникам
- подписка на все источники
- подписка на выбранный источник
- просмотр активных подписок
- отписка от всех источников
- отписка от выбранного источника
- автоматический запуск `parser.py`
- автоматическая рассылка новых новостей подписчикам

## Как работает проект

Проект состоит из двух частей:

### 1. `parser.py`
Парсер:
- получает новости из RSS-источников
- сохраняет заголовки и ссылки в SQLite-базу `news.db`
- создает таблицы `sources` и `headlines`, если их нет

### 2. `tg_bot.py`
Бот:
- работает с Telegram
- читает новости только из базы данных
- хранит пользователей и подписки
- запускает `parser.py` автоматически
- раз в заданный интервал проверяет новые записи и делает рассылку подписчикам
